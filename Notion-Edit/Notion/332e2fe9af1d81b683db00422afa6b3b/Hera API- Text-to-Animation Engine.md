> **Hera is a code-based animation generation for precise, editable motion graphics.** Hera uses LLMs to generate animation code rather than diffusion models. This enables accurate text rendering, precise data visualization, and post-generation editing.
---
# Technical Limitations of Diffusion Models
Diffusion models work by adding/removing noise over many steps. This causes:
- Text distortion and illegibility
- Geometric shapes warping (charts, tables)
- Color drift from brand specifications
- No component-level editing after generation
These limitations make diffusion unsuitable for content requiring accuracy.
---
# Code-Based Architecture
Hera generates animation code instead of pixel manipulation:
## Accuracy
- Text remains as actual text elements
- Charts maintain precise values
- Tables use proper grid structures
- Vector-based shapes stay aligned
## **Brand Consistency**
- Hex color values preserved
- Font families maintained
- Exact spacing and positioning
- Reusable brand kits
## **Editability**
- Individual elements remain accessible
- Text content is editable post-generation
- Timing is adjustable via parameters
- Export as components or rendered video
