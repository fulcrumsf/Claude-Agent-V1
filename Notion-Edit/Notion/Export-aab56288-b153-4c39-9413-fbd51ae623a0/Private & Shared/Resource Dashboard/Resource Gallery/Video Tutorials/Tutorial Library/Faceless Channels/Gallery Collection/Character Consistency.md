> I have uploaded a picture that contains two different characters, even if the characters have similarities, they are clearly unique. I will use this image as a reference in Midjourney, but to ensure that Midjourney doesn’t get confused, I need you notice all of their differences seen in the uploaded image, and make sure your prompt treats them as very distinct.  So, I want you to write prompts about  the two characters in a compelling scene together, I want the scenes to be interesting and dynamic and showcase the characters' personalities.
> 
> Each character will need to be vividly described, in a way that includes everything about them that is in the reference image, this could take six or seven lines of crafted prompt per character. But since prominence is decided in the order they appear in the prompt, I want you to write the prompt in a way that balances that prominence accordingly. Where you define the most prominent and notable thing about each character first, including where they are and what they each are doing.
> 
> The prompts will need to be carefully written so that there is no confusion whatsoever which trait belongs to which character, and at the same time, have descriptions for each one layered throughout the prompt back and forth so (assuming that’s how the scene works) neither character overshines in prominence compared to the other. You will essentially be introducing the style, then switching off between description of one character and then the other, back and forth for about 8 lines, (8-12 words each) in the context of scene, then ending with two lines of style description details. By the end of the prompt you should have described everything about each character, including what they are wearing, their eye color, and any details about their demeanor. Each description should be detailed enough that there is no way for Midjourney to misinterpret how to use the reference I shared with you.
> 
> They should be described in the same style as the reference I sent you – so ensure that style is clear in every prompt. Each prompt should be entirely self-contained (as in, you cannot assume something you explained in the first prompt will be understood in the second fully describe each character every time), but feel free to repeat some description from one prompt to the next you don’t need to reword something if it is well-written.
> 
> Please create four prompts that are different interpretations on exactly what the characters are doing and interacting to make a compelling image.
