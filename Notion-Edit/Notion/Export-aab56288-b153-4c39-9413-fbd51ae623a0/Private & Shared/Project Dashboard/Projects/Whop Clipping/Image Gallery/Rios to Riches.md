# **Content Creation Viral Strategy Masterclass**
Tue, 24 Jun 25
### **Core Components of Viral Hooks**
- Every viral video requires 3 elements:
    
    - Hook
    
    - Story
    
    - Call to action
    
- Hooks must include 2-3 of these components:
    
    - Text-based hook (captions)
    
    - Visual-based hook (what viewers see)
    
    - Audible-based hook (what viewers hear)
    
- Must capture attention in first 3 seconds
- Videos need at least 2 out of 3 hook components to perform well
### **7 Proven Hook Types**
- Negative Hooks
    
    - Point out problems/mistakes
    
    - Example: “If you workout like this, you’re killing your gains”
    
- Question-Based Hooks
    
    - Leverage curiosity and necessity
    
    - Target audience pain points
    
    - Example: “Would raising tariffs make things more expensive?”
    
- False Statement/Clickbait Hooks
    
    - Contradict common beliefs
    
    - Popular in politics/controversial topics
    
    - Used heavily by Mr Beast
    
- Social Proof Hooks
    
    - Leverage others’ success/experiences
    
    - Use higher status names/brands
    
    - Example: Reference to Mr Beast generating 100k+ views
    
- Urgency/Scarcity Hooks
    
    - Create time pressure
    
    - Emphasize limited availability
    
    - Example: “The banks are crashing - here’s what to do”
    
- Personal Connection Hooks
    
    - Focus on relatable experiences
    
    - Address specific pain points
    
    - Example: CB Writes Copy’s storytelling approach
    
- Comparison Hooks
    
    - Contrast multiple elements
    
    - Highlight benefits/differences
    
    - Example: “US vs China” content
    
### **Advanced Hook Strategies**
- Combine multiple hook types for maximum impact
- Examples of successful combinations:
    
    - Social proof + curiosity + visual hooks
    
    - Negative + question-based + curiosity hooks
    
    - Social proof + negative + credibility hooks
    
- For clippers:
    
    - Find content that allows for multiple hook types
    
    - Identify unique angles within saturated niches
    
    - Focus on amplifying existing content with strategic hooks
    
    - Look for footage that naturally contains hook elements
    
    - Prioritize quality over quantity when selecting clips
