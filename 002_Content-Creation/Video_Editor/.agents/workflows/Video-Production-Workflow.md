# Video Production Workflow

## End-to-End Process

This workflow defines the complete pipeline from research to publishing across all of Tony's YouTube channels and social platforms.

## Stage 1: Research & Ideation

### 1.1 Topic Selection
- Identify topic based on:
  - Content calendar strategy
  - Audience demand (comments, analytics)
  - Affiliate/monetization opportunity
  - Cinematic style fit
- Document in brand content strategy

### 1.2 Research Phase
- Gather 5–10 authoritative sources
- If factual content required → Use NotebookLM protocol (see 000_Skills/NotebookLM-Protocol.md)
- Create research document or NotebookLM notebook
- Store in 004_Research/reference-material/[topic]/

### 1.3 Hook & Structure
- Define primary hook (curiosity gap, number, question, surprise)
- Map narrative structure (opening, rising action, climax, resolution)
- Identify cinematic style alignment
- Document in creative brief

## Stage 2: Scriptwriting

### 2.1 Script Development
- Write full script with:
  - Shot descriptions
  - Voiceover timing
  - On-screen text callouts
  - Audio cues
- Reference NotebookLM notebook if available for fact-checking
- Target length: [channel-specific; e.g., 8–12 min for main channel]

### 2.2 Script Review
- Self-review for clarity, pacing, engagement
- Fact-check against NotebookLM or source materials
- Optimize hooks and transitions
- Finalize and version control

## Stage 3: Asset Creation

### 3.1 Visual Assets
- Identify required visual elements:
  - Stock footage/images
  - Animated sequences
  - B-roll shots
  - Graphics/text overlays
- Generate or source:
  - **Generated:** Use model selection logic from 000_Skills/Cinematic-Styles.md
  - **Stock:** Research premium stock libraries
  - **Custom:** Commission or create if unique needed

### 3.2 Audio Assets
- Record voiceover (if not using AI voice)
- Select music/background track
- Create or source sound effects
- All audio stored in project assets folder

### 3.3 Graphics & Text
- Design lower-thirds, callouts, transitions
- Create thumbnails (testing 3–5 variations)
- Design channel-specific branding elements
- Ensure brand guidelines compliance

## Stage 4: Composition & Editing

### 4.1 Edit in Remotion (or primary editor)
- Build timeline with:
  - Video sequences
  - Audio tracks
  - Text overlays
  - Transitions
- Pacing: Target [X] cuts per minute (brand-specific)
- Color grading: Apply cinematic style color profile

### 4.2 Review & Iteration
- Watch full cut for narrative flow
- Check audio sync and levels
- Verify all graphics are legible
- Color consistency check
- Final export at platform specifications

## Stage 5: Platform Optimization

### 5.1 YouTube
- **Title:** Hook + keyword research
- **Description:** 
  - First 150 chars: Hook + main value prop
  - Links to affiliate/products
  - Timestamps if >5 min
  - Social handles
- **Thumbnail:** A/B test variations
- **Tags:** Relevant keywords (brand, topic, long-tail)
- **Cards/Playlists:** Link to related videos
- **Premiere vs. Publish:** Decide based on content urgency

### 5.2 Shorts/TikTok/Instagram Reels
- Extract 15–60 second clips from main video
- Optimize vertically (9:16 aspect ratio)
- Add captions (use auto-captions, then refine)
- Hook must hit within 2 seconds
- Use music from platform libraries (rights-cleared)

### 5.3 Cross-Platform Scheduling
- Publish main video on primary platform first
- Stagger shorts across TikTok, Instagram, YouTube Shorts
- Use publishing automation (e.g., Blotato) if available
- Schedule based on audience timezone/activity

## Stage 6: Publishing & Promotion

### 6.1 Publication
- Publish to primary channel
- Verify video plays correctly on all devices
- Monitor first 24 hours for immediate feedback
- Respond to early comments

### 6.2 Promotion
- Share on relevant social platforms
- Email list notification (if applicable)
- Pin in Discord/community (if applicable)
- Affiliate partner notification (if applicable)

### 6.3 Analytics Monitoring
- Track first 72 hours for initial performance
- Monitor watch time, retention, engagement
- Screenshot key metrics for case study
- Note early comment themes

## Stage 7: Optimization & Learning

### 7.1 Data Collection
- Compile 7-day and 30-day performance data
- Track views, watch time, engagement, revenue
- Document thumbnail CTR (if A/B tested)
- Compare to channel baseline

### 7.2 Case Study Creation
- If video meets threshold (>1000 views or >5% engagement above average):
  - Create case study using framework (see 000_Skills/Case-Study-Analysis.md)
  - Document replicable elements
  - Note what underperformed
  - File in 002_Brands/[Brand]/Case-Studies/

### 7.3 Archive & Reuse
- Archive all project files (scripts, assets, composition files)
- Link case study from brand folder
- Use learnings for future content planning
- If successful format, note for franchising

## Timeline

**Typical Duration (Main Video):**
- Research & Ideation: 1–2 days
- Scriptwriting: 1 day
- Asset Creation: 2–5 days (depends on custom assets)
- Composition & Editing: 2–3 days
- Optimization & Publishing: 1 day
- **Total: 7–14 days for standard production**

**Fast Track (Shorts/Trending Content):**
- Topic to publish: 2–4 days
- Lighter asset load
- Minimal review cycles

**Hero Content (Documentary/Long-Form):**
- Total: 30–60 days
- Includes multiple review/refinement cycles
- Custom asset creation
- Deep research integration

## Maintenance

This workflow is reviewed quarterly and updated when:
- New software/platform changes workflow
- Case study reveals more efficient patterns
- Team capacity or tools change
- Brand strategy shifts production priorities
