# Title + Hook + Description Generator

This skill produces three things: a set of title options, hook variations for the first 5 seconds, and a ready-to-post description — all tuned for maximum click-through rate and viewer retention.

---

## Core Principles

These four rules govern every output. Don't compromise on them.

**1. Curiosity over clarity**
Never fully explain. The title's job is to open a gap in the viewer's mind, not close it.
- Weak: "This Animal Has Backwards Legs"
- Strong: "This Animal Shouldn't Exist"

The viewer should think *"wait — what does that mean?"* not *"oh, I get it."*

**2. Simple language wins**
Write at a 5th–8th grade reading level. No jargon, no complex phrasing. The title should be understood in under one second on a phone screen.

**3. One idea only**
Each title carries exactly one concept. The moment a title tries to do two things, it loses both.

**4. Speed of understanding**
If someone has to read it twice, it failed. Every word must earn its place.

## Title Structures

Use these as starting frameworks, then make them specific to the topic:

| Structure | Template | Example |
|-----------|----------|---------|
| **Why** | Why {subject} Is {unexpected outcome} | Why Echidnas Are Aliens |
| **Shouldn't Exist** | This {thing} Shouldn't Exist | This Animal Shouldn't Exist |
| **You Didn't Know** | Everything You Didn't Know About {subject} | Everything You Didn't Know About Crows |
| **Weird Fact** | The Weirdest {thing} Ever Found | The Weirdest Mammal Alive |
| **Question** | What Is Inside {object}? | What Is Inside a Tardigrade? |
| **Shock Fact** | This {subject} Has {impossible trait} | This Fish Punches Faster Than a Bullet |

**Title rules:**
- 4–10 words
- No punctuation clutter (avoid colons, semicolons, ellipses in titles)
- No filler words ("really", "literally", "actually")
- Must work on mobile — test it at small size mentally

Always generate **5 title options** — different structures, same topic — so the user can pick or combine.

---

## Hook Structures (First 5 Seconds)

The hook is the spoken or text opening that plays in the first 5 seconds. It must create tension immediately and lead directly into the video without any setup or intro.

| Type | Pattern | Example |
|------|---------|---------|
| **Contradiction** | State something impossible, then confirm it's real | "This animal shouldn't exist... but it does." |
| **Escalation** | Start normal, then flip | "This looks like a normal fish. Until you see what it does to glass." |
| **Fact Drop** | Drop a wild fact, then hint there's more | "This animal has 3 hearts. Two of them stop when it swims. And that's not even the weird part." |
| **Question Hook** | Ask something the viewer can't answer without watching | "What if I told you this creature has been alive since before the dinosaurs?" |
| **Reveal Hook** | Tease that something is unexplained or secret | "Scientists still can't explain what this animal does at night." |

**Hook rules:**
- 1–2 sentences maximum
- No "Hey guys, welcome back" — start IN the story
- Must create tension, not resolve it
- The hook ends where the video begins, not where it explains

Always generate **3 hook variations** — different types, same topic.

## Description Structure

Descriptions serve two purposes: SEO discovery and continued curiosity for viewers who read before clicking.

```
LINE 1 — Hook repeat (one sentence, mirrors the title's curiosity gap)
LINES 2–3 — Context (slightly more information, still doesn't give it all away)
LINE 4 — Open loop (keeps curiosity alive, implies there's more)

[blank line]

SEO keywords woven naturally into 1–2 sentences

[blank line]

#hashtag1 #hashtag2 #hashtag3 (short-form only)
```

**Platform rules:**
| Platform | Length | Style |
|----------|--------|-------|
| YouTube | Under 150 words | Full structure above, SEO-heavy |
| TikTok / Reels | 3–5 lines | Hook + open loop + hashtags only |
| Facebook | 3–5 lines | Hook + context + hashtags |

Never stuff keywords — write naturally and let the keywords appear where they fit.

## How to Generate Output

When triggered, ask for (or extract from context):
- **Topic** — what the video is about
- **Subject** — the main subject/character/thing
- **Key anomaly or hook fact** — the single most surprising or unexpected element
- **Tone** — curiosity/mystery (default), dark/intense, funny, educational
- **Platform** — YouTube, TikTok, Reels, or all

If inside a video editor project with a channel bible available, read the bible first for:
- Voice and narrator rules (words to avoid, preferred phrasing)
- Channel-specific title conventions
- Audience age and content restrictions

Then produce:

```
TITLES (5 options — different structures):
1.
2.
3.
4.
5.

HOOKS (3 variations — different types):
1.
2.
3.

DESCRIPTION:
[full platform-ready description]
```

## Example

**Input:**
- Topic: Echidna biology
- Subject: Echidna
- Key anomaly: backwards legs, lays eggs, mammal that doesn't follow mammal rules
- Tone: curiosity/mystery
- Platform: YouTube + TikTok

**Output:**

```
TITLES:
1. Why Echidnas Are Aliens
2. This Animal Shouldn't Exist
3. The Weirdest Mammal Alive
4. This Breaks Evolution
5. Nature Got This One Wrong

HOOKS:
1. This animal shouldn't exist... but it does.
2. It looks completely normal. Until you see its legs.
3. Scientists still can't fully explain why this mammal exists.

DESCRIPTION (YouTube):
This animal shouldn't exist.

The echidna has backwards legs, lays eggs like a reptile, and hasn't
changed in 200 million years. It breaks every rule evolution is
supposed to follow.

And that's just the beginning.

Discover the strangest mammal on Earth — animal facts, weird animals,
bizarre wildlife, nature science.

DESCRIPTION (TikTok/Reels):
This animal shouldn't exist... but it does.
Backwards legs. Lays eggs. 200 million years old. Unchanged.
And that's just the start.
#animals #facts #wildlife #nature #weird
```

## Final Goal

Every output should make the viewer think: *"Wait... what is that?"*

The measure of success is not how well the title describes the video — it's how badly the viewer needs to click to resolve the tension it created. CTR, hook retention (first 5 seconds), and description click-through are the metrics that matter.
