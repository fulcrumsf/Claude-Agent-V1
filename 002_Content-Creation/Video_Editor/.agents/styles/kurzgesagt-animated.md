# Style: Kurzgesagt Animated

## Overview
Kurzgesagt ("In a Nutshell") style is a fully animated explainer aesthetic built around flat vector characters, a warm-to-cool color palette, and smooth motion that makes complex topics feel approachable. Everything is illustrated — no real footage.

---

## Known Techniques (to be detailed via interview)

- **Flat vector characters** — expressive illustrated figures with simple anatomy and bold outlines
- **Scene transitions** — whip pans, smash cuts, zoom-outs that reveal larger context
- **Icon-driven information** — concepts represented as animated icons rather than text
- **Consistent color palette** — warm background tones (deep navy, burnt orange) with bright accent pops
- **Timeline / scale reveals** — zooming out from human scale to cosmic scale (or vice versa)
- **Particle effects** — dots, sparkles, and bursts used for emphasis and transitions
- **Bold typography** — clean sans-serif labels that appear in sync with narration
- **Loop animations** — background elements subtly animating to keep frames alive
