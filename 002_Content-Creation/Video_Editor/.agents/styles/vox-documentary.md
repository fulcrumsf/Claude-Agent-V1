# Style: Vox Documentary

## Overview
Vox-style is a 2D motion design documentary aesthetic made famous by Vox Media, Johnny Harris, and Kurzgesagt. It blends real archival assets (photos, maps, newspapers) with flat graphic design and animation to create a visually dense, essay-like feel.

---

## Known Techniques (to be detailed via interview)

- **Newspaper / document zoom-in** — slow push into a real historical document or photograph
- **Paper cutout characters** — archival figures given articulated limbs, animated as flat 2D puppets
- **Map reveals** — historical maps with animated border drawing, region color fills, directional arrows
- **Kinetic typography** — bold text cards that animate in sync with narration
- **Layered collage** — mixing photos, illustrations, and graphics into a single moving frame
- **Split-screen timelines** — parallel events or before/after comparisons
- **Data visualization** — animated charts and graphs that build as narrator speaks
- **Colorized archival footage** — B&W footage blended with color grading or graphic overlays
