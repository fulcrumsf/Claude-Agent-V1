## THUMBNAIL SYSTEM

### Style
- Dark backgrounds, neon accents
- Split-screen comparisons common (AI Tool A vs Tool B)
- Terminal/workflow screenshot overlays for automation content
- The AI output (generated video, image, automation result) as central visual
- Bold white text with neon outline — verdict or tutorial topic

### Layout
- Comparison: split screen with tool labels
- Review: hero AI output + bold verdict
- Tutorial: workflow diagram or before/after + step indicator

### Rules
**DO:**
- Show the actual AI output or automation result — the work is the hook
- Use "vs" comparisons — performs strongly
- Neon color accents in overlays
- Be specific — "n8n + Claude API" beats "AI Automation"

**DON'T:**
- No talking-head thumbnails — show the tool or output
- Don't be vague — specificity wins in this niche
- No generic "AI tools" roundup aesthetics
