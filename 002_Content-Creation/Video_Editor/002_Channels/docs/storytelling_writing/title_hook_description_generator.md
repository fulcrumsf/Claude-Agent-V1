# TITLE + HOOK + DESCRIPTION GENERATOR

## (Claude Code Skill -- Universal Content System)

------------------------------------------------------------------------

# PURPOSE

This system generates: - High-CTR titles - Strong opening hooks -
SEO-friendly descriptions (YouTube + TikTok + IG + FB)

Designed to work across: - Educational content - Storytelling -
Commentary - Facts / curiosity content - Any niche

------------------------------------------------------------------------

# CORE PRINCIPLES

## 1. CURIOSITY \> CLARITY

Do NOT fully explain. Create a knowledge gap.

Bad: "This Animal Has Backwards Legs"

Good: "This Animal Shouldn't Exist"

------------------------------------------------------------------------

## 2. SIMPLE LANGUAGE WINS

-   5th--8th grade reading level
-   No complex phrasing
-   No jargon

------------------------------------------------------------------------

## 3. ONE IDEA ONLY

Each title = ONE clear idea

------------------------------------------------------------------------

## 4. SPEED OF UNDERSTANDING

Viewer should understand in \<1 second

------------------------------------------------------------------------

# TITLE GENERATOR

## STRUCTURES

### 1. "WHY" STRUCTURE

Why {subject} Is {unexpected outcome}

Example: Why Echidnas Are Aliens

------------------------------------------------------------------------

### 2. "THIS SHOULD NOT EXIST"

This {thing} Shouldn't Exist

------------------------------------------------------------------------

### 3. "YOU DIDN'T KNOW"

Everything You Didn't Know About {subject}

------------------------------------------------------------------------

### 4. "WEIRD FACT"

The Weirdest {thing} Ever Found

------------------------------------------------------------------------

### 5. "QUESTION"

What Is Inside {object}?

------------------------------------------------------------------------

### 6. "SHOCK FACT"

This Animal Has {impossible trait}

------------------------------------------------------------------------

## TITLE RULES

-   4--10 words
-   Avoid punctuation clutter
-   Avoid filler words
-   Must trigger curiosity

------------------------------------------------------------------------

# HOOK GENERATOR (FIRST 5 SECONDS)

## STRUCTURES

### 1. CONTRADICTION

"This animal shouldn't exist... but it does."

------------------------------------------------------------------------

### 2. ESCALATION

"This looks normal... until you see this."

------------------------------------------------------------------------

### 3. FACT DROP

"This animal has backwards legs... and that's not even the weirdest
part."

------------------------------------------------------------------------

### 4. QUESTION HOOK

"What if I told you this animal breaks evolution?"

------------------------------------------------------------------------

### 5. REVEAL HOOK

"Scientists still can't explain this."

------------------------------------------------------------------------

## HOOK RULES

-   1--2 sentences max
-   Must create tension
-   Must lead directly into video

------------------------------------------------------------------------

# DESCRIPTION GENERATOR

## PURPOSE

Used for: - YouTube description - TikTok caption - Instagram caption -
Facebook post

------------------------------------------------------------------------

## STRUCTURE

### LINE 1 (HOOK)

Repeat curiosity

Example: This animal shouldn't exist.

------------------------------------------------------------------------

### LINE 2--3 (CONTEXT)

Explain slightly

Example: The echidna has backwards legs, lays eggs, and doesn't follow
normal evolution rules.

------------------------------------------------------------------------

### LINE 4 (OPEN LOOP)

Keep curiosity alive

Example: And that's just the beginning.

------------------------------------------------------------------------

### SEO BLOCK

Include keywords naturally: - animal facts - weird animals - nature
facts - wildlife documentary

------------------------------------------------------------------------

### HASHTAGS (SHORT FORM)

#animals #facts #wildlife #nature #weird

------------------------------------------------------------------------

## DESCRIPTION RULES

-   Keep under 3--5 lines for short form
-   Keep under 150 words for YouTube
-   Avoid keyword stuffing
-   Write naturally

------------------------------------------------------------------------
