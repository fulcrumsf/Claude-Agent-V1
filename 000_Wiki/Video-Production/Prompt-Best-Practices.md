## Summary

Comprehensive prompting guide for the video editing agent across all supported kie.ai models. Opens with a model-selection decision matrix that maps scenarios — cinematic 16:9 wildlife, fast iteration drafts, image-to-video animation, multi-shot subject consistency, vertical 9:16 short-form, transitions between known frames — to the optimal model (Veo 3.1 Quality/Fast, Kling 2.6/3.0 I2V, Wan 2.6 I2V, Sora 2 T2V/Pro Storyboard, Nano Banana 2).

It then enumerates universal prompt principles followed by model-specific guidance covering aspect ratios, motion descriptors, audio handling, reference-image conventions (Kling element references, Nano Banana up to 14 reference images), and how to invoke specialized features like first-and-last frame transitions and multi-shot storyboard mode.

Use this before writing any scene prompt: pick the model from the matrix, consult the channel's style file for visual rules, and apply the model-specific structural conventions in this guide to maximize output quality.
