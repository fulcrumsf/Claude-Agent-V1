## Summary

Defines the 6 production methodologies used across the multi-channel video content engine. These are treatment categories (visual technique, composition, aesthetic) rather than content categories — each style maps to a specific narrative purpose and AI model selection.

The 6 cinematic styles:
1. **Archival B&W Documentary** — grayscale, 1960s-90s aesthetic, used for historical credibility
2. **3D Animated Synthetic** — clean geometric visualization, educational, bright synthetic colors
3. **Satellite/Map Visualization** — Google Earth style, topographic overhead, info overlays
4. **HD Underwater Camera** — cinematic submersible footage, depth gradient, bioluminescence
5. **National Geographic Wildlife** — pro wildlife cinematography, shallow DOF, warm colors
6. **Scientific Infographic** — data-driven visualization, high-contrast diagrams

Core production rules: image-first workflow (generate stills via Nano Banana 2 on kie.ai before extending to motion), text labels always added in Remotion (never image model), creature accuracy validated against real reference images, all timing schedules built against narrator beat sheet (audio is master clock). Default platform is kie.ai (30-70% cheaper than fal.ai). Companion files include MODEL_SELECTOR.md, MODEL_CATALOG.json, and reference frames per style folder.
